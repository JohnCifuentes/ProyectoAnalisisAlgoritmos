¿Cómo se calcula?

Se usa el algoritmo de Pearson implementado manualmente en src/similarity/pearson_correlation.py (Req. #2).

  • Se calculan retornos diarios porcentuales (sin pct_change()).
  • Se alinean las fechas comunes entre cada par de activos.
  • Se elimina cualquier par de valores nulos antes del cálculo.
  • Se aplica la fórmula 2-pass: media → desviaciones → covarianza.

La matriz es simétrica (rij = rji), con diagonal = 1. Se calculan 190 pares únicos para 20 activos.

Interpretación

  • r ≈ +1 Los activos se mueven en la misma dirección. Alta diversificación con uno solo.
  • r ≈ −1 Movimiento opuesto — cobertura natural (hedge).
  • r ≈ 0 Sin relación lineal — buen candidato para diversificar un portafolio.

Limitación: Pearson mide únicamente correlación lineal. Relaciones no lineales no son capturadas.
